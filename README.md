# Order Summary Card UI

This is a simple and responsive **Order Summary Card** created using HTML and CSS. It is a front-end project designed to showcase a stylized purchase summary card, including an image, product description, pricing section, and payment options.

## 🖼️ Features

- Header image
- Title and short description
- Pricing and subscription box
- “Proceed to Payment” button
- “Cancel Order” link
- Stylish layout with gradients and shadows
